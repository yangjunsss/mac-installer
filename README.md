# mac-installer

You can install repeated and configurate the $choice variable.
```bash
curl -sSL -k https://raw.githubusercontent.com/yangjunsss/mac-installer/master/install.sh
```

Default configs is:

```bash
choice=(python on
  go on
  java on
  git on
  google-chrome on
  iTerm2 on
  SublimeText3 on
  Slack on
  Aton on
  Docker on
  Kubernetes on
  Shadowsocks off
  )
```
